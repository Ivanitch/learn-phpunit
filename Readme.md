# Learning PHPUnit

## Install project
``` 
git clone git@github.com:Ivanitch/learn-phpunit.git .
``` 
## Run project
``` 
// Install packeges
composer install

// Run test
vendor/bin/phpunit

// Or
composer test
composer test-docs
 ```
